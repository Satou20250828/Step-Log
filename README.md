## サービス概要
- 【対象】習慣化に何度も失敗してきた人／完璧主義で挫折しがちな人
- 【課題】「毎日続けなければならない」というプレッシャーにより、行動できなかった日を失敗と捉え、習慣そのものをやめてしまうこと
- 【提供価値】毎日継続を前提とせず、「できた日だけを記録する」ことで、行動をゼロにしない状態をつくる

---

## コンセプト
- コンセプト概要：行動の継続ではなく、「再開できる状態」をつくる低ハードル習慣ログ
- 大切にしている考え方：継続できないことを失敗とせず、少しでも行動した事実を肯定する
- 想定ユーザー：習慣化アプリで何度も挫折した経験があり、連続記録や未達成表示にストレスを感じやすい人
